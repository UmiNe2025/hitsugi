# 退避: M23工場で汚染された老い姿/幼子の歩行スプライト(144枚)

walke_*(老い姿)/ walkc_*(幼子)は、M23工場のNegativeプロンプトに
`human, person, figure` が含まれていた時期に生成され、全数が人物ではなく
金色の抽象紋様(輪の中の人型紋・紋章柄・装飾タイル柄)になっている。

当主が月齢18超(elder)になるとダンジョン/郷の主人公がこの紋様で描画される
実害(2026-07 ユーザー報告「マップ上の金色人型」)が出たため public/ から退避した。
エンジンは設計済みフォールバック(成人姿 walk_ へ退避)で正しく描画される。

復帰条件: 修正済みNegativeプロンプト(kanji/glyphs追加・human抑制なし)で
factory再生成し、単体の老武者/幼子が写っていることを目視確認してから
public/img/sprites/ へ戻すこと。
