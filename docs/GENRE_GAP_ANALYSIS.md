# ジャンル研究 × 灯継ぎ ギャップ分析(2026-07-06)

俺屍系・世代交代RPG・ローグライトの「作り方の定石」をWebリサーチ(6系統・出典多数)し、本作の実装状況と照合。**結論: 本作はジャンル標準のコア体験をほぼ全て実装済み**。真のギャップは少数で、多くは「バランス/コアループに触れる(要ユーザー在席)」か「素材が要る(別途生成)」。

## B. 真のギャップ(未実装/薄い)と対応区分

### (a) 今回コード実装(純表示・コアループ非改変=無人ライブでも安全)
- **G1. 目標階層パネル(短期/中期/長期の務め提示)** ← 最重要。研究の「density of goals / goal-layering / Pyramid(Base/Retention/Superfan)」に直結。新規〜中盤の「次に何を目指すか」を常に明示。現状は綴ヒント1行のみ。**→ 今回実装**。

### (b) 要ユーザー在席(store/バランス/コアループに触れる=無人ライブでは避ける)
- **G2. 段階難易度(Heat/Ascension型)**: narrativeMode 2択 → 3〜N段階の自己選択難度。newGame改+ease テーブル拡張。研究Top15 #6。バランスが動くため在席時に。
- **G3. 状態異常システム(毒/麻痺/睡眠等)**: 現状 buffs は atkUp/defUp のみ。battle.ts コアに状態異常レイヤ追加。俺屍標準だが、本作は継足/弱点/連携で代替中。要設計判断。
- **G4. 図鑑コンプの節目報酬**: 各図鑑100%/50%で一度きり奉燈付与(Zeigarnik/collection sets)。store フック要。
- **G5. デイリー挑戦(ローカル)**: seed=日付の固定ダンジョン+自己ベスト記録(localStorage)。バックエンド不要で実装可だが、遠征フローへの seed 注入が要る。中規模。

### (c) 要素材(後日生成 → docs/ASSET_NEEDS.md 参照)
- 状態異常アイコン、実績/称号バッジ絵、デイリー挑戦バナー、追加ボス/敵変種絵、章間CG増、エンドゲーム演出絵 等。

## C. 研究から得た設計インサイト(今後の指針)
1. **目標は3層重ねる**(セッション/この代/千年紀)。常に「今すぐの理由」と「大きな理由」を両立させると離脱が減る(density of goals, Clash of Clans型の隠れた長期連鎖)。
2. **失敗を前進に**: Hades型「死んでも物語/知識が進む」。本作はNG+/形見/宿敵で既に実現。宿敵の"remember"感(再来で強化)は良い実装。
3. **難易度は上下両方向に**: Heat(上)とGod Mode(下)。本作は語り部モード(下)のみ。上方向の腕試しが手薄(G2)。
4. **収集はドラマを伴わせる**: Pokédex批判=単なるチェックリストは埋まらない。本作の図鑑は縁起/宿敵で物語を伴うのが良い。節目報酬(G4)を足すと更に強い。
5. **デイリー/シード共有**は低コスト高リテンション(Rupeckは週末実装)。ただしリーダーボードはバックエンド要。ローカル自己ベストなら静的サイトで可(G5)。
