# 『灯継ぎ -HITSUGI-』没入・継続コンテンツ追加監査 M45

**監査日**: 2026-07-23  
**対象**: `main` HEAD `b8d78ae` / M43・M44公開後  
**位置づけ**: 次に追加する内容の優先順位を決める。runtime実装、画像生成、commit、push、deployは対象外。

## 1. 結論

追加した方がよい内容はある。ただし、神・敵・装備・事件をさらに増やすのではない。

次に必要なのは、既存の人物・神・地域・宿敵・形見を、`選択 → 記憶 → 再会 → 次代で回収`へ変える短い連作である。現状は神180柱、敵579体、装備810点、事件282件を持ち、量では十分である。一方、M43で追加した「今代の約束」は設定時と継承時の間が空き、神の縁は主に費用・台詞・立ち絵差分、地域事件は一回完結、宿敵は15%の再遭遇に依存している。

推奨するM45の核は次の三点である。

1. **今代の約束を、生涯の途中で二度響かせ、継承時に決算する。**
2. **家族・神・宿敵に、次の世代でも思い出せる一つの記憶を残す。**
3. **地域を、単発事件の箱ではなく、痕跡から主戦へ至る短い怪異譚にする。**

## 2. 現行実装から見える空白

| 既存の強み | 現在の止まり方 | 次の接続 |
|---|---|---|
| M43の三つの約束 | `generationVow`は設定後、主に継承sceneで一度参照される | 生涯途中の反響と、達成・未完の事実回収 |
| 家族会話・血縁連携 | 会話相手は抽選され、関係の事実が永続記録されない | 一組だけの出会い・共闘・看取り三拍 |
| `godAffinity`、契り台詞、MAX御影、星籤重複救済 | 数値・費用・画像差分はあるが、世代をまたぐ関係史がない | 露出上位12柱の神縁三幕 |
| 40地域、282事件、地域縁起、39主 | 多くの事件が単発で、主の攻略と因果が薄い | 四地域の痕跡→選択→主戦の連作 |
| 宿敵の犠牲者・地域・強化 | 再会が15%抽選で、討伐後は年代記一行へ退く | 痕跡予告、再戦差分、討滅済み因縁 |
| 形見の代数・初所有者・討伐数、宝具録 | 系譜収集は見えるが、プレイヤーが「家宝」を選び語る面がない | 三品までの家宝額装と由来 |
| local journey metrics | save-localで外部送信なし | 実ユーザーgateの基準値取得 |
| ブラウザ探索 | `DungeonRun`はsave対象外 | checkpointまたは重複なしの安全精算 |

## 3. 推奨バックログ

### P0-01 今代の約束「三つの灯」

**体験**: 約束を選んだ直後、条件に近づいた時、継承時の三回だけ短く返す。達成だけを正解にせず、未完も次代の言葉になる。

**最小実装**:

- optionalな`vowProgress`へ、既存事実から最大3件だけ記録する。
- `血脈を守る`は後継準備・全員帰還、`常夜を破る`は未踏地・主戦、`名を忘れない`は家譜・形見・人物事績から導出する。
- 生涯途中の反響は同一月最大1件、1場面10秒以内。
- 継承文を`果たした / 届かなかった / 次代へ託した`で変える。戦力報酬は付けない。

**測定**: 初見8名中6名が先代の約束と結果を説明できる。一世代5名中4名で、継承文が選んだ約束に対応する進捗事実を2件以上、誤分類なく回収する。

**見送り条件**: 月次チェックリストや取り返し不安が生じる、強制sceneが同一月1件を超える、未達が罰に見える場合は縮小する。

### P0-02 一族二人の「記憶の糸」

**体験**: 最初は家祖と最初の子の一組だけを、出会い、共闘、看取りの三拍で描く。誰とでも上がる絆ゲージは作らない。

**最小実装**:

- optionalな`relationshipMemories`を人物IDの順序非依存pair keyで保存する。
- 共有帰還、血の呼応、負傷者を連れ帰った、灯細りに立ち会った、の4事実だけを候補にする。
- 新しい強制scene枠は作らない。既存の`kizuna / hosori / daily`選出を事実参照型へ置き換え、看取りか継承の既存scene内で過去の一件を呼び戻して家譜に一文残す。
- 能力補正、好感度周回、全人物への長編量産は行わない。

**測定**: 初見8名中6名が、一組の人物名と関係を言える。看取り後に6/8が「誰の死だったか」を説明できる。

**見送り条件**: 既存scene総数またはdeferred queueを増やす、半数以上が即skipする、人物の組合せによる文意矛盾が1件でも出る、初継承中央値45分を悪化させる場合は頻度と対象を縮める。

### P0-03 冒頭を三頁と最初の意思表示へ縮約する

**体験**: 説明を読み切ってから遊ぶのではなく、最初の60秒以内に「この家へ何を遺したいか」を選び、自分の一族として始める。

**最小実装**:

- 現在11beatある導入を、寿命、星契り、常夜の三beatへ圧縮する。
- 非戦力の三択を一度だけ置き、最初の`generationQuestion`の文言へ反映する。
- 家訓、能力、M43の約束には影響させない。既存`cg_prologue.png`を使い、新規画像は作らない。

**測定**: 起動から最初の意思表示中央値60秒以内。初見8名中6名が八季の命、星契り、目的を説明でき、10分以内の契り→誕生6/8を維持する。

**見送り条件**: 設定理解6/8を割る、Home到達時間が現行より悪化する場合は、意思表示だけを残して説明量を戻す。

### P1-01 星神の記憶「縁の三幕」— まず12柱

**体験**: 星札を画像一覧ではなく、再会したい人物へ変える。`初所持 / 初契り / 次世代で再会`を一度ずつ残す。

**最小実装**:

- 露出上位12柱のみで試す。星札を操作可能にし、逸話、現在の縁、授けた子、次の一幕を表示する。
- 既存`personality / desc / pactLines / portrait / MAX御影`を使う。
- 新規札の星の文は10秒以内。重複時は縁3・MAX到達の時だけ新しい記憶を出す。
- 能力報酬や限定神は追加しない。

**測定**: 新規札取得後に5/8が詳細を開く。取得した神の名前か人物像を初見5/8が覚える。次世代まで到達した5名中3名が、同神との再契りまたは別神を選んだ理由を説明できる。30匿名save以上を得た後だけ再契り率を参考値として比較する。

**見送り条件**: 12柱で詳細閲覧20%未満、再契り率に差がない、強さだけで一極集中する場合は180柱へ展開しない。

### P1-02 四地域の「怪異三幕」＋地域主の攻略文法

**体験**: 四つのmacro biomeから一地域ずつ選び、`痕跡を見つける → 対応を選ぶ → 主戦で意味が返る`短編にする。地域主はHPの多い敵ではなく、覚えて攻略する存在にする。

**最小実装**:

- 既存事件・地域縁起・背景を再編集し、新規文章は各地域3〜6短文までに制限する。
- 進行の単一情報源は既存の`loreFrags → traceIntel → bossMikiri`とし、並行する`RegionalThread` saveや第二の見切り値を作らない。
- pilot地域は`hotarubi_no_kubochi / chochin_zaka / hoshimukuro_tani / hoshikui_no`の四つへ固定し、4 macro biomeとtier 1〜4を一つずつ代表させる。
- 四主へ2〜3手の署名patternと、手動・オート共通の兆しを追加する。
- 地域の選択は主戦の予告または安全な対処知識へ返すが、必須正解や限定報酬にしない。
- 戦果には`受けた / 崩した / 強手前に止めた`の勝因を最大2行表示する。

**測定**: 適切対処で残HPまたは被害が固定攻撃より20%以上改善。初見6/8が危険手と対処を説明し、痕跡を見た8名中4名が自発的に同地域へ再訪する。

**見送り条件**: 四主で差を説明できなければ39主へ展開しない。初見殺し、手動専用報酬、オート弱体化になる案は不採用。

### P1-03 宿敵の「狩り札」と討滅済み因縁

**体験**: 敗北を損失で終わらせず、次代が回収したくなる未完の物語にする。

**最小実装**:

- 宿敵へ遭遇回数、逃走回数、最後に見た月をoptional保存する。
- 三月に一度以下の非modalな痕跡をHome/出立に一つ表示し、再戦前に前回との差分を一文出す。
- 討伐後も家譜の`討滅済み因縁`へ残し、既存の仇討ち形見と結ぶ。

**測定**: 経験者6/8が翌日も宿敵名または犠牲者を想起。痕跡後の自発的同地域再訪率を取得する。

**見送り条件**: `理不尽 / しつこい`または宿敵痕跡を理由にした中断が2/8を超える場合は頻度を下げる。

### P1-04 「前回の灯」とDungeon中断安全性

これは追加コンテンツを遊び続けられるようにする基盤である。

- 6時間以上空いた時だけ、前回・今・次を最大3行でTitleまたはHomeに表示する。主CTAは一件、新報酬・通知・streakなし。
- 階移動、宝箱、事件、戦闘決着後にDungeon checkpointを作る。完全復元が危険なら、複製0の一度きり安全精算を先行する。
- 階段面へ現在/全階、隊最低HP、灯、主層、確定済み戦利品を出す。

**測定**: Continueから最初の意味ある行動60秒以内。4 checkpoint phaseで完全復元または安全精算100%、複製0。

### P2-01 「家宝の間」— 三品だけ額装する

**体験**: 810点の収集から、プレイヤー自身が選んだ三つの物語へ焦点を移す。

**最小実装**:

- 現存/記録済み形見から三品まで額装する。初所有者、代数、討伐、関係する約束・宿敵を既存事実から表示する。
- Homeには一品だけ、家譜には三品を置く。戦力補正を付けず、いつでも差し替え可能にする。
- 一枚画像として保存する場合も、個人情報や外部送信を行わない。

**測定**: 一世代テスト5名中3名が自発的に一品を額装し、その由来を説明できる。

**見送り条件**: 最強装備の固定枠になる、差し替えに損失がある、収集率を焦らせる場合は実装しない。

## 4. 測定契約

現行`journeyMetrics`は初回旅程9点だけを記録し、以下の候補を直接測れない。実装時は外部送信なしのoptional・one-shotなlocal eventを先に追加する。

- `intro_choice / vow_echo / vow_resolution`
- `relationship_recall`
- `god_card_open / god_generation_revisit`
- `region_trace_seen / region_trace_revisit / boss_counter_explained`
- `nemesis_trace_seen / nemesis_region_revisit`
- `resume_first_meaningful_action / dungeon_resume / heirloom_frame`

値はevent key、季、経過秒、固定enum、件数だけとし、人物名、神名、自由文、端末IDを入れない。n=8/5のpilotでは統計的な改善率を主張せず、無介助完遂、説明可能人数、重大摩擦の有無で方向性を判定する。割合比較は30匿名save以上を得た後の参考値に限定する。

## 5. 実装順とgate

### Phase 0 — 既知gateを先に閉じる

1. 現行公開版で初見8名・一世代5名の基準値を取得する。
2. M45 local event辞書とQA exportを用意する。外部送信はしない。
3. Dungeon checkpointまたは複製0の安全精算を先に閉じる。

この段階では新しい物語内容を入れない。既存版の基準値と中断安全性がないまま、複数の魅力施策を同時投入しない。

### Phase A1 — 冒頭だけを単独pilot

1. 冒頭三頁＋最初の意思表示だけを変更する。
2. 設定理解と10分以内の契り→誕生を現行基準と比較する。

### Phase A2 — 約束だけを単独pilot

1. 今代の約束「三つの灯」だけを追加する。
2. 初継承を一度通し、約束に対応する事実回収と所要時間を検証する。

### Phase A3 — 既存家族sceneの置換pilot

1. 家祖と最初の子一組だけの「記憶の糸」を、既存`kizuna / hosori / daily`枠の置換として追加する。
2. `deferred`純増0と初継承中央値を検証する。

A1〜A3は同時投入しない。各段階で初回30分または初継承中央値45分を悪化させた場合は、次へ進まない。

### Phase B — 再開導線

1. 前回の灯。

### Phase C — 再会と攻略

1. 神縁12柱。
2. 四地域怪異三幕＋四主。
3. 宿敵の狩り札。

### Phase D — 合格した型だけ拡張

神12柱・四地域で指標を満たした型だけを増やす。最初から180柱・40地域へ量産しない。

## 6. 共通受入と禁止事項

- 同一月の強制全画面sceneは既存物語を含め最大1件。
- 既存`deferred` queueの件数を純増させない。新しい記憶は既存scene枠の置換または既存継承scene内へ統合する。
- 全戦闘オートと手動の報酬差0を維持する。
- 新しい通貨、期間限定、連続日数、行動力、通知呼び戻し、未達罰を作らない。
- 新規save項目はoptional、migration冪等、破損時は既存進行へ縮退する。
- PC1280/mobile390で主CTA一件、横overflow 0、外側click/Escape/focus復帰を守る。
- 初見8名・一世代5名の外部gateを機械testで代替しない。

## 7. 今は追加しないもの

- 神、敵、装備、地域、事件、辞世の追加量産。
- 180柱すべての長編個別シナリオ。
- 新しい独立ミニゲームや追加通貨。
- ガチャ限定必須戦力、日課、期間限定、連続ログイン報酬。
- 数値だけの好感度ゲージ、関係能力補正、最強家宝ランキング。
- 実ユーザーgate前の大規模画像生成。

## 8. 推奨判断

次の実装missionは、まず**Phase 0の基準値・local計測・Dungeon中断安全性**を閉じる。その後、A1、A2、A3を別々に評価する。追加コンテンツの最優先は「今代の約束の決算」である。これはM43で作った選択を捨てずに深め、世代交代RPGとしての独自性へ最短で届く。
